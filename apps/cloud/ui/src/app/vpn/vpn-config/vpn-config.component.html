OpenVPN Server Configuration

Device tunnel IPs are allocated from this subnet OpenVPN server UDP/TCP listen port (default 1194) openvpn(8) management socket on 127.0.0.1 Mounted from secret volume at runtime

Server PKI: CA + server certs generated at image build time (per-build CA, xpmile pattern). ca.key is NOT in the image — mounted from iot-ca-key secret volume. Per-device client certs are generated by iot-cloudd at provision time using the CA key.
