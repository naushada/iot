Bootstrap PSK (commissioning)

Generate the 32-byte pre-shared key for the bootstrap handshake, then copy the Serial Number and the generated key into the cloud-ui provisioning page. The key is stored write-only and shown here only once.

Enable commissioning mode (iot.dev.mode) to generate and store the BS PSK.
